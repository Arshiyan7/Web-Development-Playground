Counter App — Information

What This App Does

This Counter App lets you increase, decrease, and reset a number with a clean and minimal interface. It’s built for beginners who want to practice DOM manipulation and event handling in JavaScript.

Features

  • Increment the counter
  • Decrement the counter
  • Reset to zero anytime
  • Simple, fast, responsive UI

How to Use

  • Tap the + button to increase the number.
  • Tap the – button to decrease it.
  • Click the reset icon to return to 0.
